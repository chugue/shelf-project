# 👉 프로젝트 주요 기능 정리
* #### 🟨 Naver OAuth 인증
* #### 🟨 Cosmus Epub을 활용한 읽기 모드
* #### 🟦 Record를 활용한 DTO 핸들링
* #### 🟦 JWT 인증 
* #### 🟦 Base64 이미지 송수신
* #### 🟦 Port One을 활용한 정기결제
* #### 🟦 ChartJS를 활용한 매출현황
* #### 🟨 RiverPod 상태관리
* #### 🟨 Dio 통신
* #### 🟦 RestDoc을 활용한 API 문서 + 통합테스트
* #### 🟦 Gitflow 브랜치 전략 활용
